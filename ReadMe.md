# Real Estate Sales Performance Dashboard  
**📅 Project Duration: December 2025**

---

## 🏢 Project Overview  
This project focuses on building a comprehensive **Power BI dashboard** to analyze real estate sales and rental performance across major Indian cities such as **Bangalore, Mumbai, Pune, New Delhi, and Nagpur**.  

The dashboard delivers meaningful insights into property trends using clean, structured, and well-processed data.

---

## 🎯 Purpose of the Project  
The primary objective is to understand **rental behavior, pricing patterns, and property distribution** across cities.  

It enables stakeholders to make informed decisions through **interactive visualizations and key performance metrics**.

---

## 📊 Key Insights Covered  

- Analysis of **total properties, average rent, and total area**  
- City-wise comparison of **rental distribution**  
- Evaluation of **BHK types and furnishing status**  
- Identification of **high-demand and high-return property segments**  
- Insights into **property categories and trends**

---

## 🧹 Data Cleaning Process (Power Query)

The dataset was transformed and cleaned using Power BI’s **Power Query Editor**:

1. **Source** – Imported raw dataset into Power Query  
2. **Promoted Headers** – Converted first row into column headers  
3. **Changed Data Types** – Assigned correct data types (text, number, date)  
4. **Removed Duplicates** – Eliminated duplicate records  
5. **Removed Blank Rows** – Cleaned null or empty entries  
6. **Duplicated Columns** – Created copies for safe transformations  
7. **Reordered Columns** – Organized columns logically  
8. **Split Columns (Position)** – Divided fixed-length values  
9. **Split Columns (Delimiter)** – Separated combined fields using delimiters  
10. **Handled Null/Error Values** – Replaced missing and erroneous data  
11. **Removed Unnecessary Columns** – Kept only relevant fields  
12. **Renamed Columns** – Improved clarity and readability  
13. **Sorted Rows** – Structured data for better analysis
