## Project Details:
---
This is a dashboard project using Power BI, and It consist urban area-wise rent calculations and how much rent each type of BHK Houses contains. Analyzing trends to determine which cities had high rent prices and which kinds of BHK apartments are mostly chosen can help identify trends. Although we can identify real estate growth from this analysis, we can also easily identify which cities consistently have high average rents. Got insights regarding the overall total property insights by city. The common rent average for all cities and the sum of areas and rates are presented. Using these insights can identify rent trends, set rent prices easily, and can sell an estate at the right price at the right time. Found overall sums of bedrooms, bathrooms, balconies, and furnishing, so can identify which cities have rich category and poor category and then modify the right rent price.
